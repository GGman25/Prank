# Просто модуль для приколов.
**Установка последней версии через [PyPi](https://pypi.org/project/fileprank/)**:
```python
pip install fileprank
```
___

**Использования:**
```python
prank.create()
```
Создает 100 txt файлов на 1 гигабайт
___
```python
prank.clear()
```
Удаляет все txt файлы из директории
